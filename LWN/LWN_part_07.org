- [[https://lwn.net/Articles/754162/][Subinterpreter support for Python]]: Eric Snow's work on PEP 554 aims to expose Python subinterpreters in the standard library, enabling multiple isolated interpreters per process with separate GILs to improve multicore performance without breaking C extension compatibility.
