* Minimal and Standard BASIC

An early standardization effort (ECMA-55). There was a Standard/Full BASIC standard (ECMA-116)
that never caught up after Microsoft BASIC took over the market.

** Implementations

- [[https://buraphakit.sourceforge.io/BASIC.shtml][John's ECMA-55 Minimal BASIC Compiler]]

- [[https://sourceforge.net/projects/bwbasic/][bwBASIC]]:  implements a large superset of the ANSI Standard for Minimal BASIC (X3.60-1978)
  and a significant subset of the ANSI Standard for Full BASIC (X3.113-1987) in C. Recent versions
  (since 2.20) aim to emulate various BASIC dialects.
