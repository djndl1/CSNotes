* History

** CTSS

- Used privileged ATTACH call to switch to recipient's file directory
- Added messages to user's PRIVATE mode mailbox file on disk
- Messages limited to 2592 BCD characters (one disk record) due to scarce disk space
- File locking prevented interference between readers/writers
- LOGIN command would print "YOU HAVE MAIL BOX" at login
- Features: user-to-user, asynchronous, file-based, persistent, secure
